Part 1

Once that is done we are to add up the total number of increases. This is fairly simple, and the following code does that through a simple recursive function and list destructing. 1 List destructuring is really one of my favorite bits of Clojure's syntax. Yes, it's common, but it's also really useful. We take the first two elements of the list, and compare them to check if they have increased, returning 1 if they have, 0 if they have not. Then we check if the list is empty, if it is we simply return, but if it is not we sum with the rest of the list. 

(defn count-increases
  [[a b & rst]]
  (let [rval (if (< a b) 1 0)]
    (if (empty? xs)
      rval
      (+ rval (count-increases (cons b rst))))))

The second problem is the generation of a three measurement sliding window. To solve this we make a slight modification by adding some further elements in the destructuring bind. Once that is done it's smooth sailing. 

(defn count-average-increases
  [[a b c d & rst]]
  (let [rval (if (< (+ a b c) (+ b c d)) 1 0)]
    (if (empty? rst)
      rval
      (+ rval (count-increases (concat [b c d] rst))))))
